--------------------------------------------------------------------------
MP3 decoding
--------------------------------------------------------------------------
MP3 files are decoded using the MP3 decoder bundled with JUCE. The MP3
patents expired in 2017, so MP3 decoding is patent-free.


--------------------------------------------------------------------------
Windows Media Foundation
--------------------------------------------------------------------------
Extraction of the audio track from video/container files (MP4, MOV, AVI,
M4A and similar) uses Microsoft Windows Media Foundation, a component of the
Windows operating system, accessed through its public API. No Microsoft
components are redistributed with this product.
